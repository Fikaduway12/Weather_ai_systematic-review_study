## Overview

This repository contains supplementary materials for the systematic review titled **"Artificial Intelligence Applications in Weather Forecasting: A Systematic Review."** The review synthesizes 50 studies published between 2015 and 2026, examining AI techniques, application domains, data sources, evaluation methods, and implementation challenges in weather forecasting.

## Key Findings Summary

- **Publication trend:** >70% of papers published 2024–2026, with peak in 2025 (38%)
- **Publication type:** Journal articles dominate (84%), conference papers (16%)
- **AI techniques:** Machine Learning (82%), Deep Learning (54%), Hybrid Models (24%)
- **Most used models:** Random Forest (25 papers), LSTM (24 papers), CNN (16 papers)
- **Application domains:** General Weather Forecasting (52%), Disaster & Early Warning (46%), Agriculture (36%)
- **Data sources:** Ground Stations (68%), Curated Databases (48%), Satellite (38%), Reanalysis (32%)
- **Evaluation metrics:** Regression metrics (74%), Classification metrics (34%), Forecast verification (24%)
- **Critical challenges:** Data quality (78%), Technical-methodological (76%), Domain-specific (46%), Socio-technical (32%)

## Column Definitions (Appendix A)

| Column | Description |
|--------|-------------|
| **Paper ID** | Unique identifier (S1–S50) for cross-referencing |
| **Author(s)** | Primary authors of the study |
| **Year** | Publication year |
| **Title** | Full title |
| **URL** | Web link to the article |
| **Publication Type** | Journal or Conference |
| **Country** | Country of first/corresponding author's affiliation |
| **Meteorological Variable(s)** | Categories of atmospheric variables predicted (Atmospheric State, Dynamic/Kinematic, Hydrological/Water Cycle, Radiation/Energy/Composition) |
| **Application Type** | Domain of application (e.g., Disaster & Early Warning, Agriculture, General Weather Forecasting) |
| **AI Technique** | Broad category of AI used (e.g., Machine Learning, Deep Learning, Hybrid) |
| **Model(s) Used** | Specific AI models employed (e.g., Random Forest, LSTM, Pangu-Weather) |
| **Prediction vs. Decision Support** | Classification as "Prediction," "Decision Support," or "Both" |
| **Dataset/Input Type** | Origin of data used (e.g., Ground Stations, Satellite/Remote Sensing, Reanalysis) |
| **Data Source** | Specific repositories or networks from which data were obtained |
| **Sample Size / Time Span / Spatial Coverage** | Size and scope of the dataset used |
| **Sample Size Category** | Relative size classification: Small, Medium, Large, or Ambigous |
| **Evaluation Metrics** | Categories of quantitative metrics used (e.g., Regression & Continuous Variable, Classification & Categorical) |
| **Key Findings** | Summary of the study's most important results |
| **Limitation Category** | Classification of limitations (e.g., Spatial & Geographic Constraints, Temporal & Extreme Event Limitations, Data Quality) |
| **Challenges Reported** | Categories of challenges reported (e.g., Data-related, Technical/Methodological, Socio-technical, Domain-specific) |

## Quality Categories (Appendix B)

| Category | Score Range | Description |
|----------|-------------|-------------|
| **High Quality** | 8–10 points | Methodologically rigorous with clear reporting |
| **Moderate Quality** | 5–7 points | Adequate but with some limitations in reporting or design |
| **Low Quality** | 0–4 points | Significant methodological or reporting limitations |
