# Appendix A: Summary of Included Studies

This appendix provides a detailed summary of the 50 studies included in the systematic review. The data were extracted from each primary study to characterize the study design, methodology, artificial intelligence techniques, application domains, data sources, evaluation metrics, key findings, and reported challenges.

---

## Column Definitions

| Column | Description |
|--------|-------------|
| **Paper ID** | Unique identifier (S1–S50) for cross-referencing |
| **Author(s)** | Primary authors of the study |
| **Year** | Publication year |
| **Title** | Full title |
| **URL** | Web link to the article |
| **Publication Type** | Journal or Conference |
| **Country** | Country of first/corresponding author's affiliation |
| **Meteorological Variable(s)** | Categories of atmospheric variables predicted |
| **Application Type** | Domain of application |
| **AI Technique** | Broad category of AI used |
| **Model(s) Used** | Specific AI models employed |
| **Prediction vs. Decision Support** | Prediction, Decision Support, or Both |
| **Dataset/Input Type** | Origin of data used |
| **Data Source** | Specific repositories or networks |
| **Sample Size / Time Span / Spatial Coverage** | Size and scope of the dataset |
| **Sample Size Category** | Small, Medium, Large, or Ambigous |
| **Evaluation Metrics** | Categories of quantitative metrics used |
| **Key Findings** | Summary of most important results |
| **Limitation Category** | Classification of limitations |
| **Challenges Reported** | Categories of challenges reported |

## Sample Size Category Definitions

| Category | Criteria |
|----------|----------|
| **Small** | ≤1,000 samples OR ≤10 stations OR ≤5 years |
| **Medium** | 1,000–100,000 samples OR 10–100 stations OR 5–30 years |
| **Large** | >100,000 samples OR >100 stations OR >30 years |
| **Ambigous** | Sample size not clearly specified or ambiguous |
