# Appendix B: Quality Assessment of Included Studies

This appendix presents the quality assessment results for all 50 studies included in the systematic review. The assessment was conducted independently by two reviewers using a customized critical appraisal tool developed specifically for AI-based meteorological forecasting research.

---

## Quality Assessment Criteria

Each study was evaluated against 10 criteria, with each criterion scored as "Yes" (1 point) or "No" (0 points). The total possible score ranges from 0 to 10.

| # | Criterion | Description |
|---|-----------|-------------|
| 1 | Research Questions | Were the research questions, aims, or objectives clearly stated? |
| 2 | Geographical Context | Was the geographical scope and meteorological context clearly defined and appropriate? |
| 3 | Study Population | Was the study population (e.g., meteorological stations, geographic region) clearly defined and appropriate for the research aims? |
| 4 | Sample Size | Was the sample size (e.g., temporal span, number of stations) adequate and justified for the AI/ML techniques employed? |
| 5 | Model Description | Was the AI/ML model clearly described, including the specific algorithms used, architecture, and hyperparameters? |
| 6 | Variables Defined | Were the meteorological variables (inputs) and prediction targets (outcomes) clearly defined? |
| 7 | Performance Metrics | Were appropriate performance metrics reported for the specific prediction task (e.g., RMSE, MAE, R² for regression; Accuracy, Precision, Recall, F1, AUC for classification)? |
| 8 | Model Validation | Was the model validated using appropriate strategies (e.g., train/test split, cross-validation, temporal holdout, external validation)? |
| 9 | Limitations | Were limitations of the study clearly discussed, including data constraints, model assumptions, and generalizability? |
| 10 | Reproducibility | Was the study described with sufficient detail to enable reproducibility (e.g., data sources, preprocessing steps, model parameters)? |

---

## Quality Categories

| Category | Score Range | Description |
|----------|-------------|-------------|
| **High Quality** | 8–10 points | Methodologically rigorous with clear reporting; meets most quality criteria |
| **Moderate Quality** | 5–7 points | Adequate but with some limitations in reporting, methodology, or design |
| **Low Quality** | 0–4 points | Significant methodological or reporting limitations; findings should be interpreted with caution |

## Notes

- Quality assessment was performed independently by two reviewers (F.W.G. and G.T.B.)
- Discrepancies were resolved through discussion until consensus was reached
- **High-quality studies** (68%) demonstrated methodological rigor with clear reporting across most criteria
- **Moderate-quality studies** (30%) had limitations primarily in sample size justification, geographical context clarity, study population definition, or performance metric reporting
- **Low-quality study** (S41, 2%) had significant limitations across multiple criteria including geographical context, sample size, model description, validation strategy, limitations discussion, and reproducibility
- The most common missing criteria across studies were: adequate sample size justification (Q4), clear study population definition (Q3), appropriate performance metrics for the task (Q7), and model validation strategies (Q8)
- Review articles (S3, S4, S5, S6, S7, S8, S12, S14, S15) inherently scored lower on criteria related to original data collection and model validation
